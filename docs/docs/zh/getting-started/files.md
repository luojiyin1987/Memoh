# 文件

每个机器人在容器里有一份**独立盘**。在 **Files** tab 里直接管。

---

## 能做什么

带工具栏、目录树、编辑器的 **FileManager** 常见能力：

### 浏览

- 面包屑上下级
- **Refresh** 看机器人刚写的文件
- **New Folder** 建目录

### 文件

- **Upload** 本机文件打进容器
- **Rename** / **Delete**（目录可递归删）
- **Download** 拉回本机

---

## 看与改

**FileViewer** 里：

- 文本类（如 `.md`、`.js`、`.py`、`.toml`）用 **Monaco** 打开，高亮、保存回写盘。
- 图（`.png` 等）可预览。

---

## 和机器人

机器人自己也能用技能、MCP 等改这些文件；**Files** 是你**肉眼查看、手改**的入口。
